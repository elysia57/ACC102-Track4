# ACC102 Track 4: AAPL vs MSFT Financial Analysis Tool
Interactive data product for financial ratio analysis using WRDS data.

## 1. Purpose & User
This tool compares the financial performance of Apple and Microsoft from 2020 to 2024.
Target users: finance students, investors, and academic users.

## 2. Data Information
- Source: WRDS Compustat
- Access Date: 2026-04-24
- Period: 2020–2024
- Companies: Apple (AAPL), Microsoft (MSFT)

## 3. Methods
- WRDS SQL data extraction
- Pandas for data cleaning and calculation
- Streamlit for interactive visualization

## 4. Key Insights
- Microsoft generally had higher ROE than Apple.
- Both companies maintained stable profit margins.
- Both firms show strong financial stability.
